# AI Resume Analyzer (ATS Scoring System)

## Description
AI-powered Resume Analyzer that evaluates resumes against job descriptions
using NLP-based keyword matching and similarity scoring to generate ATS scores
and skill-gap insights.

## Tech Stack
FastAPI | NLP | Scikit-learn | Python | PDF Parsing
